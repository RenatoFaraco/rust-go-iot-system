# Contribuindo 

Suas contribuições são muito bem-vindas! Seja corrigindo um erro, adicionando uma nova funcionalidade ou melhorando a documentações, toda ajuda é valiosa.

Para manter o projeto organizado, siga os passos abaixo.

## Como Contribuir

1. **Faça um Fork:** Crie uma cópia do repositório para a sua conta.
2. **Crie uma Branch:** Crie uma ramificação para a sua alteração (ex: `git checkout -b minha-feature`).
3. **Faça as alterações:** Edite os arquivos conforme necessário.
4. **Envie (Commit):** Faça o commit das suas alterações (ex: `git commit -m 'Adiciona nova funcionalidade'`).
5. **Envie para o repositório (Push):** Envie para a sua branch no GitHub (ex: `git push origin minha-feature`).
6. **Abra um Pull Request:** Acesse o repositório original e abra um Pull Request explicando suas alterações.

Em caso de dúvidas, sinta-se à vontade para abrir uma *Issue* ou entrar em contato.
